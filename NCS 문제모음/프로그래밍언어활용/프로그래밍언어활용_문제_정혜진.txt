1. 다음 중 연산자에 대한 설명으로 옳지 않은 것은?

1) << : 왼쪽 값을 오른쪽 값만큼 비트를 왼쪽으로 이동시킨다
2) % : 왼쪽 값을 오른쪽 값으로 나눈 나머지를 계산한다
3) && : 두 개의 논릿값 중 하나가 참이면 참(True)을 반환한고, 그렇지 않으면 거짓(False)을 반환한다
4) != : 왼쪽에 있는 값이 오른쪽에 있는 값과 다르면 참(True)을 반환한고, 그렇지 않으면 거짓(False)을 반환한다

교재 페이지 : 16~17p





2. 연산 표기법의 종류와 설명을 알맞게 연결하시오

1) 전위(Prefix) 표기법		가) 피연산자를 먼저 쓰고 연산자를 맨 뒤에 쓰는 표기법

2) 중위(Infix) 표기법			나) 연산자를 먼저 쓰고 피연산자를 뒤에 쓰는 표기법

3) 후위(Postfix) 표기법		다) 실생활에 사용되는 방식과 동일하게 쓰는 표기법

교재 페이지 : 22p





3. 다음 중 명령문에 대한 설명으로 옳지 않은 것은?

1) if문 : 조건이 참인지 거짓인지에 따라 경로를 선택한다
2) do문 : 초깃값, 최종값, 증감값을 지정하여 반복을 실행한다
3) while문 : 시작과 종료 조건을 지정하여 참인 동안에는 해당 문장을 반복해서 실행한다
4) case/switch문 : 조건에 따라 여러 개의 선택 경로 중 하나를 취하고자 할 때 사용한다

교재 페이지 : 21p


4. 다음 중 사용자 정의 자료형에 대한 설명으로 옳지 않은 것은?

1) 열거체란 괄호 안에 연속적인 값이 들어가는 자료형이다
2) 구조체란 괄호 안에 멤버 변수를 사용하는 자료형이다
3) 구조체 내부에는 멤버 변수 자료형을 마음대로 선언할 수 있지만, 멤버 함수의 작성은 불가능하다
4) 공용체는 구조체를 멤버로 사용할 수 있다

교재 페이지 : 27~28p
