운영체제 기초 활용하기(5문제)

1. 다음 빈 칸에 들어갈 말로 알맞은 것은? [p3] 
(	)는(은) 사용자로 하여금 컴퓨터의 하드웨어를 보다 쉽게 사용할 수 있도록 
인터페이스를 제공해 주는 소프트웨어이다. 하드웨어는 컴퓨터의 장치를 제어하고 데이터를 
처리 하는 중앙 처리 장치, 데이터를 저장하는 기억 장치, 외부와의 통신을 담당하는 통신 장치 
그리고 데이터 입력과 출력을 담당하는 입출력 장치 등으로 구분될 수 있다. 즉, (	)는(은)
사용자 편의성을 위한 인터페이스인 동시에 다양한 자원을 관리하는 자원 관리자이다.

2. 리눅스의 특징으로 알맞지 않은 것은? [p7]
①대부분 무료이며 지원 정책에 따라 일부 유료 서비스 제품도 있다.
②개발자, 일반 사용자가 주 사용자이다.
③오픈소스 개발이다.
④인터넷 서버, 워크스테이션 등 대형 서비스에 주로 사용된다.

3. 클라우드 서비스 용어와 설명을 알맞게 연결하시오. [p17]
IaaS ㆍ	ㆍ인프라와 운영체제뿐만 아니라 사용할 수 있는 소프트웨어까지 갖추어져 웹상의
	   로그인만으로 다양한 소프트웨어를 사용한 만큼 비용을 지불해가며 사용할 수 있는 서비스 

PaaSㆍ	ㆍ운영체제가 이미 구성되어 있는 상태에서 사용자는 데이터와 애플리케이션만 직접 관리할 수 있는 서비스

SaaSㆍ	ㆍ웹상에서 구글, 마이크로소프트, 아마존 등에서 제공하는 환경의 네트워크, 보안, 데이터 저장소, 
	   콘텐츠 딜리버리 서비스를 포함한 다양한인프라를 임대하여 이용할 수 있는 서비스 

4. 다음 빈 칸에 들어갈 말로 알맞은 것은? [p8]
운영체제를 제어하기 위한 방법은 (  a  )와 (  b  )가 있다. (  a  )는 사용자가 직접 명령어를 입력하여 
컴퓨터에게 명령을 내리는 방식이며, (  b  )는 마우스로 화면을 클릭하여 컴퓨터를 제어하는 방식이다. 
초기 윈도즈는 운영체제를 (  a  )로 설치해야 했으나, 마우스 기반의 제어 시스템인 (  b  )가 개발되며
설치 또한 화면 위주로 하게 되어 사용 빈도가 줄었다.

5. 다음 중 가상화에 대한 설명으로 알맞지 않은 것은? [p17]
①물리적인 리소스들을 사용자에게 하나로 보이게 하거나, 반대로 하나의 물리적인 리소스를 여러 개로 보이게 하는 것을 의미한다.
②사용자는 하나의 PC에 여러 개의 운영체제를 설치하여 개발되는 프로그램을 다양한 환경에서 테스트할 수 있다.
③플랫폼 가상화와 리소스 가상화로 구분될 수 있다. 
④서버 용량의 20%정도만을 사용한다.

데이터베이스 기초 활용하기(5문제)

1.  데이터베이스 관리시스템의 특징과 설명을 알맞게 연결하시오. [p25]
데이터 무결성ㆍ	ㆍ장애가 발생하였을 시 특정 상태로 복구되어야 하는 성질
데이터 일관성ㆍ	ㆍ응답 시간, 저장 공간 활용 등이 최적화되어 사용자, 소프트웨어, 시스템 등의 요구 조건을 만족 시켜야 하는 성질  
데이터 회복성ㆍ	ㆍ삽입, 삭제, 갱신, 생성 후에도 저장된 데이터가 변함없이 일정 
데이터 보안성ㆍ	ㆍ부적절한 자료가 입력되어 동일한 내용에 대하여 서로 다른 데이터가 저장되는 것을 허용하지 않는 성질 
데이터 효율성ㆍ	ㆍ불법적인 노출, 변경, 손실로부터 보호되어야 하는 성질

2. 다음 빈 칸에 들어갈 말로 알맞은 것은? [p30]
사물 또는 사건으로 정의되며 개체라고도 한다. ERD에서 (	    )는 사각형으로 나타내고 사각형 안에는 (	)의 이름을 넣는다.

3. 다음이 설명하는 데이터베이스 종류는? (		) [p25]
가장 보편화된 데이터베이스 관리시스템이다. 데이터를 저장하는 테이블의 일부를 다른 테이블과 상하 관계로 표시하며
상관관계를 정리한다. 변화하는 업무나 데이터 구조에 대한 유연성이 좋아 유지 관리가 용이하다.

①Insert
②Select

④Delete

①관계형 DBMS - 스키마 구조 필요 없음

네트워크 기초 활용하기(10문제)

1. 다음 빈 칸에 들어갈 말로 알맞은 것은? [p45]
원하는 정보를 원하는 수신자 또는 기기에 정확하게 전송하기 위한 기반 인프라를 네트워크라고 한다.
정보 전달 시에는 약속한 규칙에 따라야 하는데 이를 (	)이라고 한다.

2. OSI 7계층 중 네트워크 계층은 어느 계층인지 고르시오. [p46]
①3계층
②4계층
③5계층
④6계층

3. 다음 빈 칸에 들어갈 장비로 알맞은 것은? [p47]
(	)는 망 연동 장비이다. PC 등의 로컬 호스트가 LAN에 접근할 수 있도록 하며, WAN
인터페이스를 사용하여 WAN에 접근하도록 한다. 라우팅 프로토콜은 경로 설정을 하여 
원하는 목적지까지 지정된 데이터가 안전하게 전달되도록 한다. 

4. 다음 중 WAN에 대한 설명을 알맞지 않은 것은?. [p45]
①국가, 대륙과 같이 광범위한 지역을 연결하는 네트워크이다.
②거리에 제약이 있고, LAN보다 속도가 빠르고 에러율도 낮다.
③전용 회선 방식은 통신 사업자가 사전에 계약을 체결한 송신자와 수신자끼리만 데이터를 교환하는 방식이다.
④교환 회선 방식은 공중망을 활용하여 다수의 사용자가 선로를 공유하는 방식이다.

5. OSI 7계층 중 1계층인 물리 계층에서 사용하는 장비를 고르시오. [p47] 
①허브, 리피터
②브리지
③스위치
④라우터

6. 프로토콜의 특징으로 알맞지 않은 것은? [p53]
①단편화
②재조립
③캡슐화
④단순화

7. TCP의 특징으로 알맞지 않은 것은? [p54] 
①Flow Control, Error Control을 하지 않아 신뢰성 있는 데이터 전송에는 부적합하다.
②Flow Control 기능을 수행하여 단계별 데이터 전송 상황을 체크한다.
③논리적인 1:1 가상 회선을 지원하여 해당 경로로만 데이터가 전달되도록 한다.
④대표 서비스: FTP, Telnet, Http, SMTP, POP, IMAP 등

8. UDP의 특징으로 알맞지 않은 것은? [p54]
①연결되어 있어도 데이터를 송신할 수 있다. 단, 수신 측의 수신 여부는 확인하기 어렵다. 
②CRC 체크와 재전송 기능을 통해 신뢰성 있는 전송을 확보한다.
③하나의 송신 정보를 다수의 인원이 수신해야 할 경우 UDP를 사용한다. 
④대표 서비스: SNMP, DNS, TFTP, NFS, NETBIOS, 인터넷 게임/방송/증권 등

9. 라우팅 프로토콜의 종류 중 알맞지 않은 것은? [p62]
① RIP
② IGRP
③ ATM
④ BGP

10. 다음 빈 칸에 들어갈 방식으로 알맞은 것은? [p64]
(   a   )방식은 데이터를 일부를 송수신하여 전달 경로를 파악하고 확보한 뒤 실데이터를  
전달하는 반면, (   b   ) 전달 방식은 헤더의 주소 정보에 따라 수신 측으로 데이터를 전송한다.
