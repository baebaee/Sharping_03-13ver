<<운영체제 기초 활용하기(5문제)>>

1) 다음 중 운영체제에 대한 설명 중 틀린 것을 고르시오.

1. 운영체제는 사용자로 하여금 컴퓨터의 하드웨어를 보다 쉽게 사용할 수 있도록 인터페이스를 제공해 주는 소프트웨어이다. 
2. 최근 나오는 기억 장치는 데이터를 저장할뿐 아니라 데이터 처리도 수행한다.
3. 운영체제는 사용자 편의성을 위한 인터페이스인 동시에 다양한 자원을 관리하는 자원 관리자이다.
4. 하드웨어는 중앙 처리 장치, 기억 장치, 통신 장치 그리고 입출력 장치 등으로 구분될 수 있다.


2) 다음 중 각 운영체제에 대한 설명이 틀린 것을 고르시오.

1. 윈도즈 계열 운영체제 : 
마우스의 아이콘을 이용하여 소프트웨어를 실행시키는 편리한 인터페이스를 지원한다.

2. 윈도즈 계열 운영체제 : 
마이크로소프트사만이 수정 및 배포할 수 있으며 고객 지원이 체계적이기 때문에 보안성이 높다.

3. 리눅스계열 운영체제 : 리눅스는 유닉스의 호환 커널이다.

4.유닉스 계열 운영체제 : 
현재 서버 시장과 슈퍼컴퓨터시장에서 매우 높은 점유율을 가지고 있다.



3) 서버용 윈도즈의 특징이 맞는 것을 고르시오.

1. 사용자 관리 기능이 있다.
2. Microsoft 연동 계정을 사용할 수 있다.
3. Active Directory Domain Admin 기능이 있다.
4. 윈도즈 Server 백업 기능 외 서버에 필요로 하는 기능이 있다.
5. 윈도즈 Server 백업 기능 업데이트를 추가로 받을 수 있다. 



4) 유닉스와 리눅스의 차이점에서 모두 옳은 것으로 묶인 보기를 고르시오.

1. 비용 : 리눅스 => 대부분 유료 / 주 사용자 : 리눅스 => 개발자, 일반 사용자

2. 개발사 : 유닉스 => 대형 시스템 관리자 / 개발 배포 : 유닉스 => 대부분 사업자에 의해 배포

3. 사용자 편의 : 유닉스 : 커맨드 기반이 주였으나 GUI도 제공하는 추세, 파일시스템 제공
   개발사 : 리눅스 => IBM, HP 등

4. 사용자 편의 : 리눅스 : 기본은 Bourne Shell, 현재는 많은 Shell과 호환 가능
   사용량 : 리눅스 => 모바일폰, 태블릿 등 다양하게 사용


5) 리눅스/유닉스 계열 운영체제의 CLI 명령어에 대한 설명이 틀린 것을 고르시오.

1. 최상위 유저를 CLI 환경에서 #으로 표시하며 일반 유저를 $로 표시한다. 
2. 명령어에 대한 도움말은 -help, -hp, #man을 명령어 뒤에 붙임으로써 확인할 수 있다. 
3. #mkdir는 디렉터리를 생성하는 명령어이다.
4. -p옵션을 사용하면 하위 디렉터리까지 한 번에 생성할 수 있다.

<<네트워크 기초 활용하기(5문제)>>

1) 알맞은 용어를 적으시오.

원하는 정보를 원하는 수신자 또는 기기에 정확하게 전송하기 위한 기반 인프라를 (     1     )라고 한다.
정보 전달 시에는 약속한 규칙에 따라야 하는데 이를 (     2     )이라고 한다.


2) OSI 7 Layer의 계층에 대한 설명이 틀린 것을 고르시오.

1. 물리 계층 : 실제 장비들을 연결하기 위한 연결 장치
2. 표현 계층 : 코드 문자 등을 번역하여 일관되게 전송하고 압축, 해제, 보안 기능도 담당
3. 응용 계층 : 송신, 수신 프로세스 간의 연결 
4. 세션 계층 : 송신, 수신 간의 논리적 연결 

3) 네트워크 주요 장비에 대한 설명으로 옳은 것을 고르시오.

1. 허브는 여러 대의 컴퓨터를 연결하여 네트워크로 보내거나 
   하나의 네트워크로 수신된 정보를 여러 대의 컴퓨터로 송신하기 위한 장비이다.

2. 스위치는 소프트웨어 방식으로 처리하기 때문에 속도가 빠르며,
   브리지는 하드웨어 기반으로 처리하기 때문에 속도가 느리다.

3. 스위치는 포트들이 같은 속도를 지원하는 반면, 
   브리지는 각기 다른 속도를 지원하도록 제어할 수 있다.

4. 리피터는 디지털 신호를 증폭시켜주는 역할을 하여 신호가 약해지지 않고 컴퓨터로 수신되도록 한다.


4) 다음 중 틀린 것을 고르시오.

1. Store and Forwarding은 데이터를 전부 받은 후 다음 처리를 하는 방식이다.
2. Cut Through은 데이터의 목적지 주소만 확인 후 바로 전송 처리하는 방식이다.
3. 브리지는 Cut Through 전송 방식만을 사용한다.
4. Fragment Free는 위 두 방식의 장점을 결합한 방식이다.

5) 각 장비의 이름과 설명을 알맞게 연결한 것을 고르시오.
1. 허브 : 두 개의 근거리 통신망(LAN)을 서로 연결해 주는 통신망 연결 장치이다.

2. 스위칭허브 : LAN과 LAN을 연결하거나 LAN과 WAN을 연결하기 위한 인터넷 네트워킹 장비이다.

3. 게이트웨이 :
외부로 부터 들어오는 인터넷 라인을 연결하여 유선으로 여러대의 기계를 연결하거나 
무선 신호로 송출여 여러 대의 컴퓨터가 하나의 인터넷라인을 공유할 수 있도록 하는 네트워크 기기

4. 망(백본) 스위칭허브 : 
광역 네트워크를 커버하는 스위칭허브이다. 
예를 들어 경남권 스위칭, 부산권 스위칭 등 대단위 지역을 커버한다.
